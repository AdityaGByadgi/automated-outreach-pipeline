# Automated Outreach Pipeline

A Python-based outreach automation system.

## Workflow

1. Input company domain
2. Find similar companies
3. Find decision makers
4. Find email addresses
5. Send outreach emails
6. Safety confirmation before sending

## Technologies

* Python
* Requests
* Python Dotenv

## Run

python main.py
